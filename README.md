## Description

This is a single page resume website created using only HTML.
The purpose of this project is to practice semantic HTML structure without using CSS.

## Sections Included

* Header
* About Me
* Skills
* Experience
* Projects
* Education
* Contact Information

## How to Run

1. Download the repository
2. Open `index.html` in any web browser
